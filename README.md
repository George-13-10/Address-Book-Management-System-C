# Address Book Management System in C

A menu-driven Address Book application developed in C using structures, file handling, and modular programming to efficiently manage contact information.

---

## 📌 Overview

This project is designed to manage contact details such as name, mobile number, and email address through a command-line interface.

The application supports adding, searching, editing, deleting, and listing contacts while storing data permanently using file handling techniques.

---

## 🚀 Features

- Add new contacts
- Search contacts
- Edit existing contacts
- Delete contacts
- List all saved contacts
- Persistent data storage using files
- Menu-driven user interface

---

## 🛠 Technologies Used

- C Programming
- Structures
- File Handling
- Modular Programming
- Command Line Interface (CLI)
